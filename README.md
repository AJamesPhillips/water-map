
# Water Map UK

A work in progress concept of a UK water availability map.

Currently [hosted here](https://wikisim.org/wiki/1219)

## Getting Started

    git clone git@github.com:AJamesPhillips/water-map.git water-map-uk --recursive
    cd water-map-uk
    pnpm install
    pnpm run dev
